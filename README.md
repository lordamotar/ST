# Stoly-Sklad (ST) — Мебельный салон премиум-класса

Современная платформа для управления мебельным каталогом и онлайн-продаж в Казахстане (Семей, Астана). Проект сочетает в себе высокопроизводительный бэкенд на **FastAPI**, динамичный фронтенд на **Next.js 16** и мощную административную панель для управления бизнесом.

## 🚀 Стек технологий

*   **Backend:** Python 3.12+, FastAPI, SQLAlchemy (Async), PostgreSQL, Pydantic, JWT Auth.
*   **Frontend:** React 19, Next.js 16 (Turbopack), Tailwind CSS, Framer Motion (анимации).
*   **Database:** PostgreSQL.
*   **API:** RESTful API с автоматической документацией Swagger/ReDoc.

## ✨ Основные возможности

*   💎 **Премиальный дизайн**: Современный интерфейс с эффектами Glassmorphism и плавными анимациями.
*   📦 **Управление каталогом**: Полный CRUD товаров и категорий через админ-панель.
*   ⏱️ **Система маркетинга**: Таймеры обратного отсчета для акций, управление старыми/новыми ценами.
*   📊 **Админ-дашборд**: Статистика заказов, управление пользователями и настройками сайта.
*   📂 **Массовый импорт**: Загрузка товаров через Excel с автоматическим парсингом характеристик и дат акций.
*   ⚡ **SEO Оптимизация**: Динамические мета-теги, канонические ссылки, sitemap и JSON-LD разметка.

## 🛠️ Быстрый старт

### 1. Подготовка окружения
Создайте файл `.env` в корне проекта по примеру:
```env
DATABASE_URL=postgresql+asyncpg://user:pass@localhost/st_db
SECRET_KEY=your_secret_key_here
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=43200
```

### 2. Установка зависимостей
```powershell
# Установка зависимостей бэкенда (используя uv)
uv sync

# Установка зависимостей фронтенда
cd frontend
npm install
cd ..
```

### 3. Инициализация проекта
```powershell
# Запуск автоматического обновления и миграции
./scripts/update.ps1

# Создание администратора по умолчанию (admin/admin)
python scripts/setup_admin.py
```

## 📜 Управляющие скрипты

Все вспомогательные инструменты собраны в папке `scripts/`:

*   `./scripts/update.ps1` — автоматическое обновление: git pull, миграции, установка зависимостей.
*   `python scripts/migrate.py` — обновление схемы базы данных без потери данных.
*   `python scripts/seed.py` — наполнение базы демонстрационными данными.
*   `./scripts/deploy.ps1` — сборка фронтенда и очистка проекта перед запуском.
*   `python scripts/setup_admin.py` — создание первого суперпользователя.

## 📂 Структура проекта

*   `app/` — Сердце бэкенда (модели, схемы, API-эндпоинты).
*   `frontend/` — Клиентское приложение на Next.js.
*   `scripts/` — Утилиты для автоматизации и обслуживания.
*   `uploads/` — Директория для загруженных изображений товаров.
*   `tests/` — Набор тестов для бэкенда.
